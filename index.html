Servo & Photoresistor Project
#arduino
This is the project where I made a circuit using a photoresistor and a servo. The photoresistor detects the light level from my phone's flashlight and the servo points to the according 'weather', detecting how much 'sunlight' there is. I made this project along with Paul McWhorter from his Arduino series on Youtube. The code and the project does not belong to me, I solely built and coded it along with him to learn Arduino. All credits belong to Paul McWhorter, a great teacher!

High Torque Servos & Joystick Project
#arduino
This is the project where I taped two servos together (one moving in x-axis, one moving in y-axis, together allowing for three-dimensional movement) and controlling the servos with the help of a joystick. If you check out my github (BareMinimum34) there is also a buzzPin, which is for the push button of the joystick, acting as a 'shooter' haha. Again I coded and built this circuit along with Mr. Paul McWhorter following his Arduino series on youtube. All credit goes to him.
